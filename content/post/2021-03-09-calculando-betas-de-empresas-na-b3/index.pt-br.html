Nesse post, calculamos o coeficiente Beta do modelo CAPM de empresas de capital aberto com ações negociadas na B3.

Começamos com o download dos tickers de empresas ativas em março de 2021.

Download tickers.xlsx

A leitura dos tickers é feita com o comando read_excel, do pacote readxl https://readxl.tidyverse.org/

library(readxl)

As ações de empresas brasileiras são encontradas no yahoofinance com a adição de “.SA” aos tickers. Então, temos que adicionar esse sufixo ao nosso arquivo de tickers, criando a coluna codigo no dataframe tickers .

tickers <- read_excel("tickers.xlsx", sheet=1, col_names = TRUE)
tickers$codigo <- paste(tickers$`unique(Dados$Codigo)`,".SA",sep="")
lista <- tickers$codigo

A função tq_get do pacote tidyquant https://cran.r-project.org/web/packages/tidyquant/index.html, permite a obtenção de preços diários das ações diretamente do https://finance.yahoo.com/, para todos os nossos tickerts. É um processo que pode demorar, dependendo da qualidade da conexão com a internet. No mesmo bloco de código, transformamos os preços diários em retornos logarítmicos ajustados, que são definidos da seguinte forma:

\[LogReturns = \log(\frac{P_{t}+Proventos}{P_{t-1}})\] O Código para obter as informações de preço ajustado das ações que compõem nossa lista de tickers, bem como da Bovespa, estão nos blocos abaixo:

library(tidyquant)
library(dplyr)
log_Retornos_diarios <-   lista %>%
  tq_get(get  = "stock.prices",
         from = "2016-03-01",
         to   = "2021-02-28") %>%
  group_by(symbol) %>%
  tq_transmute(select     = adjusted, 
               mutate_fun = periodReturn, 
               period     = "daily",
               type = "log",
               col_rename = "Ra")
log_Rb <- "^BVSP" %>%
  tq_get(get  = "stock.prices",
         from = "2016-03-01",
         to   = "2021-02-28") %>%
  tq_transmute(select     = adjusted, 
               mutate_fun = periodReturn, 
               period     = "daily",
               type = "log",
               col_rename = "Rb")

Considerando que utilizaremos os dados para calcular os coeficientes betas das empresas brasileiras da B3, faremos a junção do objeto com os retornos das ações com os objetos contendo os retornos do índice IBOVESPA, usando a função dplyr::left_join.

log_Retornos_diarios_indice <- left_join(log_Retornos_diarios, log_Rb, by = c("date" = "date"))
